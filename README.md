Damka — веб-платформа для игры в русские шашки с AI-тренером, матчмейкингом и системой геймификации.
Игра доступна без регистрации. Гость может сыграть против ИИ трёх уровней сложности, после чего ему предлагается создать аккаунт и открыть полный функционал.
ИИ-противник построен на алгоритме Minimax с Alpha-Beta отсечением. Каждый уровень сложности отличается глубиной просчёта ходов. После каждой партии авторизованный игрок получает разбор от Google Gemini: что сделал хорошо, где допустил ошибку и как улучшить игру.
Для игры с другом реализован мультиплеер по ссылке через Supabase Realtime. Матчмейкинг подбирает соперника по рейтингу ELO с защитой от гонки состояний.
Система удержания включает ежедневные стрики с наградами на отметках 3, 7, 14 и 30 дней, игровую валюту за победы и магазин скинов для фигур, досок и эффектов. Лидерборд показывает топ игроков по городам.
Стек: Next.js 14, TypeScript, Tailwind CSS, Supabase, Google Gemini API, Vercel.
